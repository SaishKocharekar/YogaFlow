# 🧘 YogaFlow – AI-Powered Wellness Platform

![YogaFlow Banner](https://img.shields.io/badge/YogaFlow-AI%20Powered%20Wellness-success)

A modern full-stack wellness platform designed to help users improve their physical and mental well-being through personalized BMI analysis, AI-powered recommendations, yoga guidance, meditation resources, and wellness tracking.

## 🌐 Live Demo

https://yoga-flow-psi.vercel.app

---

## 📖 Overview

YogaFlow is a comprehensive wellness ecosystem that combines modern web technologies with personalized health recommendations.

The platform allows users to:

* Track their BMI and wellness progress
* Receive personalized AI-based wellness recommendations
* Access yoga and meditation resources
* Explore diet plans based on their health goals
* Purchase wellness-related products
* Monitor their progress over time

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/SaishKocharekar/YogaFlow.git
```

### Navigate to Project

```bash
cd YogaFlow
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server directory.

```env
PORT=5000

JWT_SECRET=your_secret_key

FIREBASE_API_KEY=your_key
FIREBASE_AUTH_DOMAIN=your_domain
FIREBASE_PROJECT_ID=your_project_id
FIREBASE_STORAGE_BUCKET=your_bucket
FIREBASE_MESSAGING_SENDER_ID=your_sender_id
FIREBASE_APP_ID=your_app_id
```
